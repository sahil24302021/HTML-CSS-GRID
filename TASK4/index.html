Questions & Answers

1. What does auto do in grid rows?

auto adjusts the row height according to the content inside it.

2. Which row grows based on content?

The row with auto value grows based on its content.

3. Can rows have different heights?

Yes, grid rows can have different heights.

4. What happens if content is very large?

The auto row increases its height to fit the content.

5. Where are grid rows useful in real layouts?

Grid rows are useful in layouts like headers, content sections, and footers.
