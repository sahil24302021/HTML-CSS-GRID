TASK 10: CSS @supports

Grid Output (Questions & Answers)

1. Why do we check browser support?

To make sure CSS features work correctly in all browsers.
2. What happens if Grid is not supported?

The browser ignores Grid styles and uses normal layout.
3. Is this needed for all projects?

No, it is mainly needed for older browsers.
4. Who uses this in real projects?

Developers use it in large websites that support many browsers.
5. What does safe CSS mean?

Safe CSS means writing styles that work even if some features are not supported.
