Task 1: CSS Grid Introduction

Grid Output (Questions & Answers)

1. What is the difference between a normal div and a grid container?

A normal div follows block layout, while a grid container uses CSS Grid to control layout using rows and columns.
2. Why does the layout look the same even after applying Grid?

Because no rows or columns are defined, Grid uses default one-column layout.
3. Are grid items created automatically?

Yes, when display: grid is applied, all direct children become grid items.
4. Can Grid work without rows or columns?

Yes, Grid works with default rows and one column even if none are defined.
5. What does Grid need to show visible changes?

Grid needs defined columns, rows, or spacing properties.
