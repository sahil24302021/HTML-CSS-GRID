Questions & Answers

1. What is the difference between a normal div and a grid container?

A normal div arranges elements using normal block layout, while a grid container uses the CSS Grid system which allows layout control using rows and columns.

2. Why does the layout look the same even after applying Grid?

Because no rows or columns are defined, the grid uses default settings, which place items one below another like normal block elements.

3. Are grid items created automatically?

Yes, when display grid is applied to a container, all its direct children automatically become grid items.

4. Can Grid work without rows or columns?

Yes, Grid works with default one column and automatic rows even if no rows or columns are defined.

5. What do you think Grid needs to show visible changes?

Grid needs defined columns, rows, or spacing properties to show visible layout changes.
