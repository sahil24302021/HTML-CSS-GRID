TASK 3: FIXED vs FLEXIBLE COLUMNS

Fixed Columns (px)

Which layout breaks faster on small screens?

Fixed layout using px breaks faster on small screens.
Why is fixed layout risky?

Fixed layout does not change with screen size and may cause overflow.
Where can fixed columns be useful?

Fixed columns are useful in dashboards or layouts with fixed width content.

Flexible Columns (fr)

Which layout adjusts automatically?

Flexible layout using fr adjusts automatically.
Which unit would you choose for a responsive website?

fr unit is better for responsive websites.
How does flexible layout behave on resize?

The layout adjusts smoothly according to available screen size.
