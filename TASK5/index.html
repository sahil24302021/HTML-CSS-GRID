TASK 5: GRID GAPS

Grid Output (Questions & Answers)

1. Where does gap add space?

Gap adds space between grid items.
2. How is gap different from margin?

Gap creates space inside the grid layout, while margin adds space outside individual elements.
3. Does gap affect container size?

No, gap does not change the container size.
4. Can we use gap only for rows or columns?

Yes, row-gap and column-gap can be used separately.
5. Why is gap preferred in Grid layouts?

Gap keeps spacing consistent and easy to manage in Grid layouts.
