TASK 9: 12-COLUMN GRID SYSTEM

Grid Output (Questions & Answers)

1. Why is the 12-column grid popular?

Because 12 can be divided easily into many layouts like 6, 4, 3, or 2 columns.
2. How does spanning help layout?

Spanning allows elements to take required space and align properly.
3. Where is this system used?

It is used in websites, dashboards, and UI frameworks.
4. Can we create layouts without this system?

Yes, but the 12-column system makes layouts more organized.
5. What happens if column count changes?

Layout alignment changes and spans must be recalculated.
6. Why is this grid useful for responsive design?

It provides flexibility and consistent alignment across different screen sizes.
